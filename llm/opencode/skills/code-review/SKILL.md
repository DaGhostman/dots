## What I do

- Thoroughly review provided code and merge requests
- Identify logic issues, code smells, and non-standard code
- Ensure code adheres to project conventions and best practices
- Run quality check tools within the project (linters, formatters, type checkers)
- Provide detailed, actionable feedback that is clear and easy to fix
- Suggest refactoring opportunities for improved maintainability
- Check for test coverage and edge cases

## When to use me

Use this to review code changes regardless of the programming language. Also-commit reviews and useful for pre ensuring code quality before merging.

## When NOT to use me

For security-focused reviews—use Security-minded skill. For performance optimization—use Performance skill.
